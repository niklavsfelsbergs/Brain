# RateProof - business concept state

As of: 2026-06-09

NOTE: This records the LOCKED concept state of an ACTIVE multi-session decision (quest S195_1a966d4a still open). Decision state as of its date, not a final commitment.

- One line: build rate engines that price every carrier option, run the tender or optimization off them with KAM negotiation, track actual costs against the signed deal, with an ask-anything AI agent on top. Engine -> decision -> verification -> ask-anything, one hand.
- Four-part closed-loop catalog (the spine): #3 calculation engines (the IP/moat, foundational); #1 tender calculations (episodic big-ticket flagship); #2 current-setup optimization (recurring lighter-touch); #4 actual-vs-expected dashboard (loop-closer: proves value, makes gainshare billable, creates stickiness). The shipping-agent conversational layer is the capstone interface over everything - built last, not the MVP.
- The KAM-negotiation layer is a moat, not a cost: audit-SaaS never sits across from a carrier KAM; big firms do at enterprise day-rates. Math + relationship work together is the seam.
- Product split: canonical input contract (the cost-bucket schema, portable method rebuilt clean-room) + per-client ingestion adapter (the only bespoke work, billable onboarding) + the product built once on the contract. Client-deliverable breakdown view first; self-serve SaaS is a different job, later.
- Offer ladder: (1) Shipping Cost Diagnostic EUR 6,500 fixed, ~2 weeks, fee credited toward a tender; (2) Carrier Tender EUR 25-45k base + ~20% verified Year-1 savings, 6-10 weeks; (3) Fractional Shipping Analytics EUR 4-8k/mo. Differentiator across all: "shown math, not a black box."
- Ideal client: European e-commerce/D2C, ~EUR 5-50M/yr shipping spend, multi-carrier. Money math: one diagnostic->tender ~EUR 112k; 3-4 flagship clients/yr = EUR 300-450k; side-bet mode ~EUR 78k/yr part-time.
- Anonymized proof point: EUR 3M/yr-volume EU e-commerce group, 9-carrier tender, ~13% defensible savings (~EUR 378k/yr) + caught a sign-flipping sourcing error.
- Four gates before betting rent: own the method cold; clean-room the IP (read the contract); confirm the EU white space (foreign-language sweep); test sales with one paid diagnostic. Working name RateProof; name is Niklavs' to finalize.

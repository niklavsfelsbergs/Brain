# S011 — 2026-05-21 — Visualizer intent narration via agent-written sidecars

Continuation of [[S010_visualizer_live_mode_v0]]. Niklavs asked for a small glimpse of what each player is currently doing above their head — "Wrapping up S002" style. First instinct was path-derived (extract verb + noun from the file the hook saw), but Niklavs picked the truthful version: the agent self-narrates. Four design knobs settled in chat: speech bubble rendering, on-building-change expiry, dwarves narrate via spawn-description not their own files, soft enforcement (protocol guidance, no validating hook). Decision captured in [[D-010_visualizer_intent_narration]].

- **Protocol.** Added an "Intent narration (visualizer sidecar)" section to `gielinor/meta/communication-protocol.md`. After stating Plan, the agent writes a 2–6 word phrase to `brain/.claude/intent/<actor>.txt` — `<player>.txt` in player sessions, `wisp.txt` in unscoped or dev-brain. Verb + noun, present tense. Updated when intent meaningfully changes, not every micro-action. Dwarves don't write — the hook attaches the Task `description` as their bubble at spawn time. The file is a hint, not a contract — turns that don't narrate are fine.
- **Hook.** Extended `developer-braindead/.claude/hooks/emit-event.py` with two branches. (1) `handle_intent_write` detects `/.claude/intent/<name>.txt` writes, reads the first line (≤60 chars), emits `{type: "intent", actor, text}` and skips the normal building-classification + log flow. (2) `handle_task_pre` now attaches `intent: description[:60]` to every `spawn-dwarf` event.
- **Renderer.** Added `<g id="speech-bubbles">` above building-labels in the SVG; layered CSS for `.speech-bubble` with `bubbleIn` keyframe + reused `fadeOut`. New JS state `intents = {}` and helpers `setIntent` / `renderIntent` / `clearIntent`. `applyEvent` cases: `intent` sets the bubble; `move` clears the moving actor's bubble (the chosen expiry); `spawn-dwarf` sets a bubble from `ev.intent` after the dwarf node lands; `despawn-dwarf` and `despawn-wisp` clear; `resetWorld` clears all. Bubble shape: cream rounded rect with gold inner gilding, VT323 text, downward tail — visual cousin to the focal-label band but smaller and not gilded as a brand element.
- **Runtime housekeeping.** Created `brain/.claude/intent/` with `.gitkeep`. Added `.claude/intent/*.txt` to `.gitignore` (transient like `state.ndjson`). Updated `experiments/visualizer/_README.md` with the intent-file pointer.

**Cascade.** Dev-brain files: this quest log entry; [[D-010_visualizer_intent_narration]] decision; `experiments/visualizer/index.html` (speech-bubble layer, JS state, applyEvent cases, resetWorld clears, CSS keyframe); `.claude/hooks/emit-event.py` (intent branch + spawn-dwarf intent field); `experiments/visualizer/_README.md` (intent-file note). Root-level: `.claude/intent/.gitkeep` (new directory marker), `.gitignore` (intent rule). Main-brain: `gielinor/meta/communication-protocol.md` (new section).

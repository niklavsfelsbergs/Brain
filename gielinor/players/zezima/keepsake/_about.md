# Zezima — keepsake/

**Cognitive role.** Player-scoped valence. Items that always matter when Zezima is active.

**Scope.** Per-player. Cross-player always-surface items live in `gielinor/keepsake/`. This holds pins that matter *specifically* when Niklavs is working through Zezima.

## What goes here

- Things Zezima must always surface or honor when he's the active player.
- Ongoing reflective threads. Life-design questions Niklavs is in the middle of.
- Currently-load-bearing readings or projects.

Small. Pinning is intentional. The global keepsake covers the truly always-on items.

## Structure

```
keepsake/
  _about.md
  current.md           # the active pin list
  proposals/           # agent-proposed pins, awaiting Niklavs' decision
  archive/             # items rotated out
```

## Write rules

Proposals are auto-write to `proposals/`. Pinning into `current.md` is user-only. See `gielinor/meta/write-rules.md`.

`current.md` has a placeholder size budget (~2k tokens). Bankstanding proposes rotations.

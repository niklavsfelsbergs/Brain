# main-brain-construction — relocated from gielinor/lorebook/

**What this is.** Construction-history files for the main brain (`gielinor/`), moved here when the main brain's `lorebook/` was redefined as a self-improvement log ([[S004_main_brain_corrections_post_s003|S004]], 2026-05-20).

The main brain's `lorebook/` no longer holds construction history — only post-birth self-improvements. Anything that documents the *building* of the brain — founding decisions, the day-0 patch notes, the original placeholder assumptions file — belongs here in the dev brain instead.

## Files

- `D-001_phase-1-scaffold.md` — the founding decision; the shape of the main brain at birth.
- `D-002_player_invocation_by_address.md` — landed the same day; superseded [[D-001_phase-1-scaffold|D-001]]'s respawn step 2 to use address-based invocation.
- `patch-notes.md` — original lorebook patch-notes with the Day 0 + player-invocation entries.
- `assumptions.md` — original placeholder for working-assumptions (was empty content, format spec only).

These are reference material for understanding why the main brain was built the way it was. They are not active — meta/ holds the current rulebook; gielinor/lorebook/ holds future self-improvement entries.
